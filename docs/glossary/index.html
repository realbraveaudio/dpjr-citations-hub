Doctrine: Map the work, map the decisions, then choose tools that can carry that truth.

Glossary

Work Map
How work actually flows, including exceptions and workarounds.
Decision Map
How judgment routes when uncertainty hits; who owns which decisions.
Tool Map
Whether tools mirror the work/decisions or force distortion.
Frankenstein Stack
A mismatched set of tools held together with manual glue.
Ownerless decision
A decision with no clear owner or boundary; it escalates by default.
Workaround
A manual step required because a tool cannot represent real work.
